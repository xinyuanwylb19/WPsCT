⚡ End-use Products Tracker

  1. 1

    Select a Data Source

    Choose a built-in example (Maine or United States), or upload your own data (see the format below).

  2. 2

    Choose Model Parameters

    Use the default parameters, or upload a custom parameter file for your region.

  3. 3

    ▶ Run Tracker

    Runs the model. Results appear in the table; the Download button activates.

  4. 4

    📊 Plot & Explore

    Use "Plot Input Data" and "Plot Carbon Stock" to view production trends and the accumulated carbon pools. Hover over charts for exact values.

📂 Input Data

The production data file must be a comma-separated (.csv) file with the following columns. All numeric values must be in kg C/yr.

Column Name Pool Description
Year-Calendar year (integer). Required. No gaps.
BiofuelBiofuelAnnual biofuel combustion (kg C/yr) - industrial + residential fuel wood.
BiocharBiocharNon-energy use biochar production (kg C/yr).
ConstructionEnd-useWood in building construction - sawnwood, structural panels (kg C/yr).
ExteriorEnd-useExterior products - railway ties, wood decking, dock materials (kg C/yr). No recycling.
HouseholdEnd-useHome application - furniture, flooring, interior millwork (kg C/yr).
Graphic PaperEnd-usePrinting & writing paper, newsprint (kg C/yr). Service half-life γ = 6 yr.
Household PaperEnd-useTissue, toweling, sanitary paper (kg C/yr). Service half-life γ = 0.5 yr. No recycling.
Other PaperEnd-usePackaging, cardboard, industrial paperboard (kg C/yr). Service half-life γ = 1 yr.
